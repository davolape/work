- Скриншот вывода команды "sudo netstat -tan"
![команда "sudo netstat -tan"](src/photo_5339463996648061209_x.jpeg) 
- Ключи -tan:
- t - показывает только TCP-соединения
- a - показывает все соединения, активные и слушащие порты
- n - выводит IP-адреса и порты в числовом виде, без попытки преобразовать в имена
- tcp - протокол соединения
- 0 0 - количество отправленных и полученных байтов которые ждут обработки
- 0.0.0.0:2022 - локальный адрес и порт, 0.0.0.0 - означает на всех интерфейсахб удаленный адрес и порт для любого подключения, а 2022 - слушающийся порт
- LISTEN - сокет слушает входящие соединения, и процесс привязан ко всем IP-адресам компа, т.е. порт доступен по любому интерфейсу
